AdvancedMapViewer改
====

## Overview
---

mapsforge のWebサイトで公開されている AdvancedMapViewer の改良

位置のブックマーク機能と簡易ジオコーディング機能を追加

## Install
---

[Google Play](https://play.google.com/store/apps/details?id=jp.gr.java_conf.ya.shiobeforandroid) からインストールできます
